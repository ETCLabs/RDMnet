◆ more_coming

bool RdmnetEptClientList::more_coming

This message contains a partial list.

This can be set when the library runs out of static memory in which to store Client Entries and must deliver the partial list before continuing. The application should store the entries in the list but should not act on the list until another RdmnetEptClientList is received with more_coming set to false.
